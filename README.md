# Mini Landing — Día 1 (Fundamentos Web)

Proyecto del día 1 del plan de estudios fullstack developer.

## Descripción
Landing page responsive construida con HTML semántico, CSS moderno (Flexbox + Grid) y Git como control de versiones.  
El objetivo fue reforzar la accesibilidad, la estructura semántica y el flujo básico de Git.

## Tecnologías usadas
- HTML5
- CSS3 (Flexbox y Grid)
- Git
